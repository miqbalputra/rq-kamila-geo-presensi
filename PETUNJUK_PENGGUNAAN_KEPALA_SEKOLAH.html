PANDUAN LENGKAP GEOPRESENSI

Sistem Monitoring Kehadiran & Kedisiplinan Berbasis GPS

1. Akses Sistem & Login

Gunakan akun khusus monitoring yang telah diberikan oleh Administrator IT untuk mengakses portal:

Admin: https://geo.griyaquran.web.id/admin
Guru: https://geo.griyaquran.web.id/guru

Sistem dapat diakses 24/7 melalui HP (Smartphone) atau Laptop.

2. Monitoring Kehadiran Real-Time

Dashboard Monitoring: Melihat persentase kehadiran hari ini secara instan (Siapa yang tepat waktu, terlambat, izin, atau sakit).

Daftar "Belum Hadir"

Menampilkan guru yang seharusnya sudah di lokasi tapi belum presensi.

Validasi Radius 20m

Kehadiran hanya bisa dicatat jika guru berada dalam radius ±20 meter dari sekolah.

3. Fitur Unggulan: Jadwal Piket & QR Code

Sistem mendukung efisiensi operasional harian melalui:

  • Cetak QR Code: Admin dapat mencetak QR Code dari menu "QR Code Presensi" untuk ditempel di tempat strategis.
  • Jadwal Piket Digital: Guru piket mendapat notifikasi khusus di aplikasi mereka.
  • Monitor Piket: Pantau siapa saja petugas piket hari ini langsung dari Dashboard.

4. Kalender & Hari Libur Sekolah

Sistem ini sangat fleksibel dalam mengatur jadwal operasional sekolah:

Otomatisasi Libur: Admin mendaftarkan hari libur nasional atau khusus sekolah. Pada hari-hari tersebut, sistem otomatis menutup fitur presensi agar statistik guru tidak dianggap "Alpha".

5. Pelaporan & Akuntabilitas (Excel)

Unduh laporan lengkap untuk keperluan rapat atau penilaian kinerja:

  1. Pilih periode (Harian, Mingguan, atau Bulanan).
  2. Filter nama guru jika ingin melihat histori individu.
  3. Klik tombol Download Excel untuk mendapatkan file siap cetak.

💡 Tips untuk Kepala Sekolah

  • Cek Dashboard Jam 07:30: Pantau kesiapan kelas sejak awal.
  • Gunakan Data untuk Reward: Berikan apresiasi bagi guru dengan tingkat keterlambatan 0 menit setiap bulannya.
  • Evaluasi Mingguan: Gunakan tren statistik untuk memantau kedisiplinan yang mulai menurun.
