───────────────────────────────────────────────────────────────
  STEP 3: CEK JADWAL DI DATABASE
───────────────────────────────────────────────────────────────

1. Login cPanel → phpMyAdmin
2. Pilih database: sobataja2_geopresence
3. Tab "SQL"
4. Query:

SELECT * FROM jadwal_piket 
WHERE hari = 'Jumat'
ORDER BY id DESC;

EXPECTED:
  ✅ Ada data jadwal untuk hari Jumat
  ✅ user_id sesuai dengan ID Pandu
  ✅ jam_piket = 02:00:00

JIKA TIDAK ADA DATA:
  ❌ Tidak ada jadwal piket untuk hari Jumat
  
SOLUSI:
  1. Login admin
  2. Menu "Jadwal Piket"
  3. Tambah jadwal untuk Pandu di hari Jumat
  4. Jam piket: 02:00

───────────────────────────────────────────────────────────────
  STEP 4: CEK USER ID PANDU
───────────────────────────────────────────────────────────────

Query di phpMyAdmin:

SELECT id, username, nama, role 
FROM users 
WHERE nama = 'Pandu' OR username LIKE '%pandu%';

EXPECTED:
  ✅ Ada user dengan nama "Pandu"
  ✅ Catat ID-nya (misal: id = 2)

Lalu cek jadwal piket:

SELECT * FROM jadwal_piket 
WHERE user_id = 2;  ← Ganti 2 dengan ID Pandu

EXPECTED:
  ✅ Ada jadwal piket untuk user_id = 2
  ✅ hari = 'Jumat'
  ✅ jam_piket = '02:00:00'
