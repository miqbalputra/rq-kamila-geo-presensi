# Panduan Admin: Mengelola Status "Hadir - Izin Terlambat"

## Kapan Menggunakan Status Ini?

Gunakan status **"Hadir - Izin Terlambat"** untuk guru yang:
- ✅ Sudah mendapat izin dari kepala sekolah untuk datang terlambat
- ✅ Memiliki alasan yang jelas (rumah sakit, mengurus anak, dll)
- ✅ Sudah melakukan presensi saat tiba di sekolah
- ✅ Keterlambatan sudah dikonfirmasi dan disetujui

## Langkah-Langkah Penggunaan

### Cara 1: Edit Presensi yang Sudah Ada

1. **Login sebagai Admin**
   - Buka aplikasi presensi
   - Login dengan akun admin

2. **Buka Menu Edit Presensi**
   - Klik menu "Edit Presensi" di sidebar kiri

3. **Cari Presensi yang Akan Diubah**
   - Gunakan filter tanggal untuk menemukan presensi
   - Cari nama guru yang akan diubah statusnya

4. **Edit Status**
   - Klik tombol "Edit" (ikon pensil) pada baris presensi
   - Form akan muncul di bagian atas halaman
   - Ubah dropdown "Status" menjadi **"Hadir - Izin Terlambat"**
   - Jam masuk dan jam pulang tetap sama (tidak perlu diubah)

5. **Tambahkan Keterangan**
   - Isi kolom "Keterangan" dengan alasan izin
   - Contoh: "Izin ke rumah sakit - sudah konfirmasi kepala sekolah"
   - Contoh: "Mengurus anak sakit - izin kepala sekolah"

6. **Simpan Perubahan**
   - Klik tombol "Update Presensi"
   - Status akan berubah dan ditampilkan dengan badge biru

### Cara 2: Tambah Presensi Manual

Jika guru belum melakukan presensi sama sekali:

1. **Buka Menu Edit Presensi**
2. **Isi Form Tambah Presensi**
   - Pilih nama guru dari dropdown
   - Pilih tanggal presensi
   - Pilih status: **"Hadir - Izin Terlambat"**
   - Isi jam masuk (sesuai jam datang sebenarnya)
   - Isi jam pulang (jika sudah pulang)
   - Tambahkan keterangan alasan izin
3. **Klik "Tambah Presensi"**

## Contoh Kasus Nyata

### Kasus 1: Guru ke Rumah Sakit
**Situasi**: Ibu Siti izin ke kepala sekolah untuk datang jam 09:00 karena harus ke rumah sakit (jam masuk normal: 07:00)

**Langkah**:
1. Ibu Siti melakukan presensi saat tiba jam 09:00
2. Sistem mencatat sebagai "Hadir (Terlambat)"
3. Admin mendapat konfirmasi dari kepala sekolah
4. Admin edit presensi Ibu Siti:
   - Status: "Hadir - Izin Terlambat"
   - Keterangan: "Izin ke rumah sakit - sudah konfirmasi kepala sekolah"
5. Simpan perubahan

### Kasus 2: Mengurus Anak
**Situasi**: Pak Budi izin datang jam 08:30 karena harus mengantar anak ke dokter

**Langkah**:
1. Pak Budi presensi jam 08:30
2. Sistem catat sebagai "Hadir (Terlambat)"
3. Admin verifikasi dengan kepala sekolah
4. Admin ubah status menjadi "Hadir - Izin Terlambat"
5. Keterangan: "Mengantar anak ke dokter - izin kepala sekolah"

### Kasus 3: Keperluan Keluarga
**Situasi**: Ibu Ani izin datang siang karena ada keperluan keluarga mendesak

**Langkah**:
1. Ibu Ani presensi saat tiba
2. Admin ubah status ke "Hadir - Izin Terlambat"
3. Keterangan: "Keperluan keluarga mendesak - izin kepala sekolah"

## Tips untuk Admin

### ✅ DO (Lakukan):
- Selalu verifikasi dengan kepala sekolah sebelum mengubah status
- Tulis keterangan yang jelas dan spesifik
- Ubah status sesegera mungkin setelah konfirmasi
- Simpan bukti izin jika ada (foto/email/chat)
- Catat jam datang sebenarnya, jangan diubah

### ❌ DON'T (Jangan):
- Jangan ubah status tanpa konfirmasi kepala sekolah
- Jangan kosongkan kolom keterangan
- Jangan ubah jam masuk menjadi jam normal
- Jangan gunakan status ini untuk keterlambatan tanpa izin
- Jangan lupa simpan perubahan

## Melihat Laporan

### Di Dashboard
- Status "Hadir - Izin Terlambat" ditampilkan dengan **badge biru**
- Dihitung sebagai "Hadir" dalam statistik
- Jam masuk tetap menunjukkan waktu sebenarnya

### Di Download Laporan
- Status muncul sebagai "HADIR - IZIN TERLAMBAT"
- Masuk dalam kolom "Hadir" di statistik
- Keterangan ikut tercetak di laporan

## Pertanyaan Umum (FAQ)

**Q: Apakah status ini mempengaruhi persentase kehadiran?**
A: Tidak. Status ini tetap dihitung sebagai "Hadir" sehingga tidak menurunkan persentase kehadiran guru.

**Q: Apakah guru bisa memilih status ini sendiri saat presensi?**
A: Tidak. Hanya admin yang bisa mengubah status ini setelah verifikasi.

**Q: Bagaimana jika guru lupa presensi sama sekali?**
A: Admin bisa menambahkan presensi manual dengan status "Hadir - Izin Terlambat" dan mengisi jam sesuai waktu kedatangan.

**Q: Apakah bisa mengubah status lama (bulan lalu)?**
A: Ya, admin bisa mengubah status presensi kapan saja, tidak ada batasan waktu.

**Q: Apa bedanya dengan status "Izin"?**
A: Status "Izin" untuk guru yang tidak hadir sama sekali. "Hadir - Izin Terlambat" untuk guru yang hadir tapi terlambat dengan izin.

**Q: Apakah perlu mengubah jam masuk?**
A: Tidak perlu. Biarkan jam masuk sesuai waktu sebenarnya untuk transparansi.
