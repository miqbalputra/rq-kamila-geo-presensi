- **Guru:** https://kelolasekolah.web.id/guru

---

## 👨‍🏫 PANDUAN CEPAT UNTUK GURU

### Cara Presensi HADIR
1. ✅ Aktifkan GPS smartphone
2. 🌐 Buka https://kelolasekolah.web.id/guru
3. 🔐 Login (username & password dari admin)
4. 🏫 Pastikan berada di sekolah (dalam radius 500m)
5. ✋ Klik tombol **HADIR** (hijau)
6. 📍 Izinkan akses lokasi
7. ✅ Selesai! Status tercatat

### Cara Presensi IZIN/SAKIT
1. Login ke sistem
2. Klik tombol **IZIN** (kuning) atau **SAKIT** (merah)
3. Isi keterangan (wajib)
4. Klik **Simpan**
5. ✅ Tidak perlu GPS

### Cara Presensi PULANG
1. Tersedia mulai jam **09:00 WIB**
2. Pastikan GPS aktif dan di sekolah
3. Klik tombol **PRESENSI PULANG** (biru)
4. ✅ Jam pulang tercatat

### Menu Guru
- **Home:** Dashboard dan tombol presensi
- **Riwayat Saya:** Lihat riwayat presensi
- **Status Rekan:** Lihat status rekan kerja
- **Statistik Saya:** Lihat statistik kehadiran

## 🔧 PANDUAN CEPAT UNTUK ADMIN

### Tugas Utama Admin
1. ✅ Kelola data guru
2. ✅ Atur jadwal piket
3. ✅ Kelola hari libur
4. ✅ Atur pengaturan sistem
5. ✅ Download laporan

### Menu Admin
- **Dashboard:** Monitoring kehadiran real-time
- **Data Guru:** Tambah/edit/hapus guru
- **Hari Libur:** Kelola kalender libur
- **Jadwal Piket:** Atur jadwal piket per hari
- **Download Laporan:** Export data ke Excel
- **Log Aktivitas:** Audit trail lengkap
- **Pengaturan:** Atur kebijakan sistem

### Pengaturan Penting
- **Jam Masuk Normal:** Default 07:20
- **Toleransi Terlambat:** Default 15 menit
- **Radius GPS:** Default 500 meter
- **Koordinat Sekolah:** Dari Google Maps
- **Mode Testing:** Aktif/Nonaktif validasi GPS
- **Terlambat Piket:** Ubah status atau warning saja

## 🌟 FITUR UTAMA

### 1. Validasi GPS
- Guru harus di sekolah untuk presensi hadir
- Radius default: 500 meter
- Mencegah presensi dari luar sekolah

### 2. Deteksi Terlambat Otomatis
- Status otomatis "Hadir Terlambat" jika lewat batas
- Keterangan berapa menit terlambat
- Toleransi bisa diatur admin

### 3. Jadwal Piket
- Admin atur jadwal piket per hari
- Guru dapat notifikasi jika piket
- Warning jika terlambat piket

### 4. Statistik Individual
- Setiap guru lihat statistik sendiri
- Persentase kehadiran otomatis
- Filter: Bulan Ini, Bulan Lalu, 3 Bulan, Tahun Ini

### 5. Laporan Excel
- Download laporan periode tertentu
- Filter per guru atau semua guru
- Data lengkap untuk administrasi

### 6. Hari Libur
- Admin kelola kalender libur
- Sistem blokir presensi di hari libur
- Weekend otomatis libur

### 7. Presensi Pulang
- Tersedia mulai jam 09:00
- Validasi GPS juga berlaku
- Data jam kerja lengkap

### 8. Status Rekan
- Guru lihat status rekan kerja
- Transparansi kehadiran
- Update real-time

## ⚠️ TROUBLESHOOTING CEPAT

### Guru Tidak Bisa Presensi
**Masalah:** "Anda berada di luar jangkauan sekolah"
- ✅ Pastikan GPS aktif
- ✅ Mendekatlah ke area sekolah
- ✅ Coba di area terbuka
- ✅ Hubungi admin jika masih error

**Masalah:** "Gagal mendapatkan lokasi"
- ✅ Aktifkan GPS di Settings
- ✅ Izinkan akses lokasi di browser
- ✅ Restart smartphone
- ✅ Coba browser lain

**Masalah:** Tombol presensi tidak muncul
- ✅ Cek apakah hari libur
- ✅ Cek apakah sudah presensi
- ✅ Refresh halaman (F5)
- ✅ Logout dan login lagi

### Admin
**Masalah:** Guru lupa password
- ✅ Login admin → Data Guru
- ✅ Edit guru → Set password baru
- ✅ Beritahu guru password barunya

**Masalah:** Presensi tidak tercatat
- ✅ Cek Log Aktivitas
- ✅ Cek koneksi internet
- ✅ Input manual jika perlu
