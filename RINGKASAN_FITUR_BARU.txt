───────────────────────────────────────────────────────────────
  FITUR 1: JADWAL PIKET GURU
───────────────────────────────────────────────────────────────

📋 ADMIN:
  • Menu "Jadwal Piket" di sidebar
  • Kelola jadwal piket per guru per hari
  • Set jam piket (default: 07:00 WIB)
  • Add, Edit, Delete jadwal

👤 GURU:
  • Badge info piket di dashboard (jika ada piket hari ini)
  • Warning khusus jika terlambat hadir piket
  • Status tetap "hadir" tapi ada notifikasi terlambat

🎯 CARA KERJA:
  1. Admin set jadwal piket untuk guru tertentu di hari tertentu
  2. Guru yang piket harus hadir maksimal jam piket
  3. Jika terlambat: Status "hadir" + Warning terlambat piket
  4. Contoh: Piket jam 07:00, hadir jam 07:15 → Warning 15 menit
