📦 UPLOAD SEKARANG (1 LANGKAH!)
═══════════════════════════════════════════════════════════════

1. Login cPanel → File Manager → public_html
2. HAPUS: index.html (lama) + assets/ (lama)
3. UPLOAD: dist/index.html (baru) + dist/assets/ (baru)
4. Clear cache (Ctrl+Shift+Del) + Hard refresh (Ctrl+F5)
5. Login admin → Cek menu "Hari Libur" muncul ✅
