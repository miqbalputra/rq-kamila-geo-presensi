═══════════════════════════════════════════════════════════════

MASALAH:
  Endpoint GET ALL dijalankan saat request ?today=1
  karena kondisi !isset($_GET['id']) = TRUE
  
  GET ALL return: array [{...}, {...}]
  GET TODAY expect: object { hari: "...", jadwal: [...] }

SOLUSI:
  Tambah kondisi !isset($_GET['today']) di GET ALL
  Sekarang GET ALL hanya jalan jika TIDAK ADA parameter today
