🎯 LANGKAH UPLOAD (SIMPLE!)
═══════════════════════════════════════════════════════════════

STEP 1: Upload Backend
  → Login cPanel → File Manager
  → Masuk ke /home/username/api/
  → Upload: api/holidays.php

STEP 2: Jalankan SQL
  → Login cPanel → phpMyAdmin
  → Pilih database
  → Tab "SQL"
  → Copy-paste isi database_hari_libur.sql
  → Klik "Go"

STEP 3: Upload Frontend
  → Login cPanel → File Manager
  → Masuk ke public_html
  → HAPUS: index.html (lama)
  → HAPUS: assets/ (lama)
  → UPLOAD: dist/index.html (baru)
  → UPLOAD: dist/assets/ (baru)

STEP 4: Test
  → Clear cache: Ctrl + Shift + Delete
  → Hard refresh: Ctrl + F5
  → Login: guru / guru123
  → Cek tampilan hari Sabtu (hari ini!)
