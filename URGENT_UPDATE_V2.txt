  🔍 CARA CEK APAKAH BERHASIL
───────────────────────────────────────────────────────────────

BUKA CONSOLE BROWSER (F12) → Tab "Console"

Expected Log (SUKSES):
  ⏳ Retry 1/3...
  ✅ Data loaded successfully: { id: 123, ... }
  
Jika muncul log ini → BERHASIL! Tampilan akan update.

Expected Log (RETRY):
  ⏳ Retry 1/3...
  ⏳ Retry 2/3...
  ✅ Data loaded successfully: { id: 123, ... }
  
Jika muncul log ini → BERHASIL! Butuh 2x retry.

Expected Log (FALLBACK):
  ⏳ Retry 1/3...
  ⏳ Retry 2/3...
  ⏳ Retry 3/3...
  ⚠️ Max retries reached, setting data manually
  
Jika muncul log ini → Pakai fallback, tapi tetap berhasil.

Error Log (MASALAH):
  ❌ Failed to load data: Error: ...
  
Jika muncul log ini → Ada masalah, screenshot dan lapor.
